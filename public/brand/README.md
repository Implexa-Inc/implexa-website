# Implexa Logo System

Generated logo variants for implexa.ai using the existing brand tokens:

- Warm dark base: `#0A0805`
- Surface: `#15110C`
- Foreground: `#FAFAF8`
- Emerald signal: `#10B981` / `#34D399`
- Flame action: `#FF8A3C`

## Files

- `implexa-wordmark-dark.svg`: wordmark for dark backgrounds.
- `implexa-wordmark-light.svg`: wordmark for light backgrounds.
- `implexa-mark.svg`: square app/social mark.
- `implexa-lockup-dark.svg`: mark plus wordmark for dark backgrounds.
- `implexa-lockup-light.svg`: mark plus wordmark for light backgrounds.
- `implexa-social-card.svg`: 1200x630 social preview artwork.
- `logo-preview.html`: local preview sheet.

The wordmark intentionally uses a dotless `ı` so the emerald signal dot can be positioned as a brand node.
